# Sistema Eletrocar – Scripts SQL

Projeto desenvolvido para a disciplina de Modelagem e Banco de Dados, com base no mini-mundo de uma oficina mecânica chamada Eletrocar.

## Objetivo

Implementar e manipular dados em um banco de dados relacional utilizando SQL (DML), a partir do modelo lógico normalizado (1FN, 2FN e 3FN).

## Tecnologias

- SGBD: MySQL 
- Ferramenta: MySQL Workbench ou pgAdmin
- Linguagem: SQL
